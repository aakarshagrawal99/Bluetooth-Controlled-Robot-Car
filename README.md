# Bluetooth-Controlled-Robot-Car
Arduino Based Bluetooth Controlled Robot Car

This project demonstrates a Bluetooth controlled robot car using Arduino Uno.

## Problem Statement
Wired control systems limit mobility and flexibility in small robotic vehicles.

## Solution
A smartphone-based Bluetooth control system using the HC-05 module.

## Components Used
- Arduino Uno
- HC-05 Bluetooth Module
- L298N Motor Driver
- DC Gear Motors
- Robot Car Chassis
- Battery Pack

## Features
- Forward, backward, left, right movement
- Wireless control using mobile app
- Beginner-friendly robotics project

## Learning Outcomes
- Arduino programming
- Motor driver interfacing
- Bluetooth communication
- Embedded systems basics
